## 70. 爬樓梯

### 題目

假設你正在爬樓梯。需要 n 階你才能到達樓頂。
<br>
每次你可以爬 1 或 2 個台階。你有多少種不同的方法可以爬到樓頂呢？

示例 1：
```
輸入：n = 2
輸出：2
解釋：有兩種方法可以爬到樓頂。
1. 1 階 + 1 階
2. 2 階
```
示例 2：
```
輸入：n = 3
輸出：3
解釋：有三種方法可以爬到樓頂。
1. 1 階 + 1 階 + 1 階
2. 1 階 + 2 階
3. 2 階 + 1 階
```

### 官方解答

#### 遞迴 (方法一)
最直觀最容易想到的方式<br>

```
func climbStairs(n int) int {
    if n == 1 {
        return 1
    }
    if n == 2 {
        return 2
    }
    return climbStairs(n-1) + climbStairs(n-2)
}
```

時間複雜度：
$O(2^n)$ <br>
空間複雜度：
$O(n)$
